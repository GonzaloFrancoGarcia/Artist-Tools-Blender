# Herramientas-Blender-Artista

## Características

* **Mirror (Espejo):** Crea simetrías rápidas en los ejes X, Y, Z con control de distancia.
* **Align (Alineación):** Alinea fácilmente los objetos seleccionados con el objeto activo.
* **Pivot (Pivote):** Mueve el origen del objeto a la **Geometría** o al **Cursor 3D** con un solo clic.
* **Selection (Topología):** Selecciona caras según su número de lados (muy útil para encontrar N-Gons).
* **Normals (Normales):** Voltea (Flip) las normales de las caras al instante.

## Cómo instalar

1.  Descarga el archivo `addon.py`.
2.  Abre Blender y ve a **Edit > Preferences > Add-ons**.
3.  Haz clic en **Install**, selecciona el archivo descargado y activa la casilla **Object: Artist Tools Addon**.
4.  Presiona la tecla **N** en la Vista 3D y busca la pestaña **"Artist Tools"**.
